# Unraid-Docker-Orchestrator
Gestionnaire intelligent de démarrage et mise à jour Docker pour Unraid. Gère l'ordre des dépendances (VPN, DB), le rollback automatique en cas d'erreur et les notifications natives. Inclus un système de "Self-Healing" contre les erreurs de copier-coller (NBSP). Solution 100% libre, gratuite et robuste pour votre serveur.
